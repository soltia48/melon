# Melon 利用規約

## 第1条（目的）

本規約は、アンノウン・テクノロジーズ株式会社（以下「当社」といいます。）が提供する前払式支払手段「Melon」（以下「本サービス」といいます。）の利用条件を定めるものです。利用者が本サービスを利用する場合には、本規約が適用されます。

## 第2条（定義）

本規約において使用する語句の定義は、次のとおりとします。

- **Melon（本残高）**
  当社が本規約に基づき利用者に対して発行する、円単位の金額についての電子的情報であって、利用者が加盟店において商品等の代金の支払いに使用することができる前払式支払手段。
- **カード**
  利用者が本残高を利用するために用いる非接触ICカード（FeliCa）であって、当社が本サービスに対応するものとして指定したもの。
- **カード識別情報**
  カードに固有の識別情報であって、当社のサーバーが相互認証を経て取得し、本残高を特定するために用いるもの。
- **利用者**
  カードを用いて本サービスを利用する方。
- **加盟店**
  当社との加盟店契約に基づき、商品等の代金の支払いに本残高を使用することができる事業者。
- **店舗**
  加盟店が本サービスを取り扱う個々の店舗。
- **端末**
  加盟店の店舗に設置され、チャージ、支払い、残高照会等を行うために用いられる機器。
- **商品等**
  利用者が加盟店から販売又は提供を受ける物品、サービス、デジタルコンテンツ及び権利等。
- **チャージ**
  利用者が対価を支払い、当社から本残高の発行を受けること。
- **仮名ID**
  当社が加盟店ごとに個別に割り当てる、利用者を特定しない識別子。

## 第3条（本規約への同意及び適用）

1. 本規約は、本サービスを利用するすべての利用者に適用されます。
2. 利用者は、チャージ、支払いその他本サービスの利用を行った時点で、本規約の内容に同意したものとみなされます。
3. 当社は、本規約を当社ウェブサイト（ https://melon.unknowntech.jp/terms ）に掲載する方法により表示します。加盟店は、その店舗において、本サービスの利用に本規約が適用される旨を掲示するものとします。
4. 本サービスの利用にあたり、利用者の会員登録、専用アプリケーションのインストール等は必要ありません。

## 第4条（対応カード）

1. 本サービスは、当社が対応するものとして指定した FeliCa カードにおいてのみ利用できます。すべての FeliCa カードで利用できるものではありません。
2. 利用者は、自己の費用と責任においてカードを用意するものとします。
3. カードの品質、故障その他カード自体に関する事項については、利用者は当該カードの発行者又は販売者との間で解決するものとします。

## 第5条（本残高の管理方法）

1. **本残高は、カードには記録されず、当社のサーバーにおいて管理されます。**
2. カードは、利用者の本残高を特定し、その正当な保有者であることを確認する（認証する）ためにのみ使用されます。利用者が端末にカードをかざすと、当社のサーバーとカードとの間で相互認証が行われ、カード識別情報により本残高が特定されます。
3. **加盟店は、カード識別情報を取得することができません。**加盟店に対しては、当該加盟店ごとに異なる仮名IDのみが提供されます。したがって、複数の加盟店が相互に照合しても、同一の利用者であることを識別することはできません。
4. 本サービスの利用には、当社のサーバーとの通信が必要です。通信できない場合、本サービスは利用できません。

## 第6条（チャージ（本残高の発行））

1. 利用者は、加盟店の店舗に設置された端末において、当社所定の方法によりチャージを行うことができます。
2. 本残高は、当社のサーバーにおいて記録された時点をもって、利用者に対して発行されたものとします。
3. チャージの対価は、当社に直接又は加盟店を通じて当社に支払われます。
4. 停電、機械故障、システムの保守点検、通信障害、不正防止その他安全管理上やむを得ない事由により、チャージを中止することがあります。

## 第7条（有効期限及び失効）

1. **本残高には、チャージごとに有効期限が設定されます。**
2. 有効期限は、当該チャージが行われた日（日本標準時。以下同じ。）の**6か月後の応当日の午前0時**とします。
3. 前項の6か月後の月に応当する日が存在しない場合は、当該月の末日を応当日とします（例：8月31日にチャージした場合、応当日は翌年2月28日（閏年は2月29日）となります。）。
4. 利用者は、第2項の時刻が到来する前、すなわち**応当日の前日まで**、当該チャージ分の本残高を使用することができます。
5. **有効期限が到来した未使用の本残高は、当該時刻をもって失効します。**失効した本残高は、以後いかなる方法によっても使用することができず、当社は、これについて払戻しその他一切の補償を行いません。
6. 有効期限は、いかなる理由によっても延長されません。
7. 利用者は、端末において、本残高及びその有効期限を確認することができます。

## 第8条（支払い（本残高の使用））

1. 利用者は、加盟店の店舗に設置された端末にカードをかざす方法により、商品等の代金の全部又は一部の支払いに本残高を使用することができます。
2. 支払いは、端末に表示された金額を利用者が確認したうえで、カードを端末にかざすことにより行われます。利用者は、端末に表示された金額及び支払後の残高に誤りがないかを確認し、誤りがあった場合は速やかに当該加盟店に申し出るものとします。
3. **利用者が有効期限の異なる複数のチャージ分の本残高を保有する場合、支払いには、有効期限が早く到来する本残高から順に充当されます。**有効期限が同一である場合は、チャージが行われた時期が早いものから順に充当されます。
4. 本残高が代金に不足する場合、当該支払いは成立しません。この場合の不足額の取扱いは、加盟店の定めに従うものとします。
5. 支払いが完了した時点で、当該支払額に相当する本残高が減算され、当該加盟店に対する商品等の代金の支払いが完了したものとします。
6. 一部の商品等については、その代金の支払いに本残高を使用できない場合があります。

## 第9条（取消し及び返金）

1. 加盟店は、商品等の返品その他の事由により、既に行われた支払いの全部又は一部を取り消し、又は返金することができます。
2. **前項による返金は、当該支払いの充当元となったチャージ分の本残高に戻されます。この場合において、当該チャージ分の有効期限は延長されません。**
3. 充当元となったチャージ分の本残高が既に失効している場合、返金された本残高は失効した状態となり、これを使用することはできません。この場合の利用者と加盟店との間の精算は、現金その他の方法により、利用者と加盟店との間で行うものとします。
4. 商品等の瑕疵、返品の可否その他利用者と加盟店との間に生じる取引上の一切の問題は、利用者と加盟店との間で解決するものとします。

## 第10条（払戻しの制限）

1. **未使用の本残高の払戻しは、次の各号に定める場合を除き、行いません。**
   1. 資金決済に関する法律その他の法令に基づき払戻しが義務付けられる場合
   2. その他当社が特に認める場合
2. 前項に基づき払戻しを行う場合、当社は、当社所定の方法により未使用の本残高を確認したうえ、当社所定の手続に従って払い戻します。この場合、当社は、正当な保有者であることの確認のために必要な情報の提供を求めることがあります。
3. **有効期限の到来により失効した本残高は、払戻しの対象となりません。**
4. 当社が正当な保有者であることを確認できない場合、又は未使用の本残高を確認できない場合、当社は払戻しを行わないことができます。

## 第11条（カードの管理、紛失及び盗難）

1. 利用者は、自己の責任においてカードを厳重に管理・保管するものとします。
2. **カードは本残高を特定する鍵として機能します。**したがって、利用者がカードを紛失し、又は盗難に遭った場合、利用者は当該カードに紐づく本残高を使用できなくなるおそれがあります。
3. 当社は、カードの紛失、盗難、破損又は故障の場合において、本残高の他のカードへの移行、再発行その他の補償を行いません。ただし、当社が特に認めた場合はこの限りではありません。
4. カードの紛失、盗難等により第三者が本残高を使用した場合に利用者に生じた損害は、利用者の負担とします。

## 第12条（禁止事項）

利用者は、本サービスの利用にあたり、次の各号の行為を行ってはなりません。

1. 違法、不正又は公序良俗に反する目的で本残高を使用すること
2. 対価の有無を問わず、本残高を第三者に譲渡し、貸与し、移転し、又は担保に供すること
3. 営利を目的として本残高を使用すること
4. カード、端末又は本サービスに関するシステムを破壊、改変すること
5. 本残高の複製を試み、又はこれに加担・協力すること
6. 偽造、変造若しくは不正に作出されたカード又は本残高を使用すること
7. マネー・ローンダリング、テロ資金供与その他の犯罪に本サービスを利用すること
8. その他本規約に違反する行為

## 第13条（本サービスの利用の中止・停止）

1. 当社は、次の各号のいずれかに該当すると認めた場合、利用者に予告することなく、本サービスの全部又は一部の利用を中止することがあります。
   1. カード又は本残高が偽造、変造若しくは不正に作出されたとき、又はその疑いがあるとき
   2. 本残高が不正に使用されたとき、又はその疑いがあるとき
   3. システムの障害、通信障害、保守点検その他の事由により本サービスの提供が困難となったとき
   4. 利用者の利用が本規約に違反し、又は違反するおそれがあるとき
   5. その他やむを得ない事由が生じたとき
2. 利用者は、カード又は本残高が偽造、変造又は不正に作出されたものであることを知ったときは、これを使用してはならず、直ちに当社に通知するものとします。

## 第14条（加盟店及び商品等）

1. 加盟店契約の締結及び終了等により、加盟店及び店舗の数は増減することがあります。
2. 当社及び加盟店は、本残高を使用することができない商品等を個別に追加又は変更することができます。

## 第15条（情報の取扱い）

1. 当社は、本サービスの提供にあたり、**原則として利用者の個人情報を取得しません。**
2. 当社は、カード識別情報並びにチャージ、支払い、返金及び失効に関する取引履歴を取得し、これを本サービスの提供、不正利用の防止、システムの保守・改善及び統計的分析の目的で利用します。
3. 当社は、加盟店に対し、当該加盟店ごとに異なる仮名IDのみを提供し、カード識別情報を提供しません。
4. 当社は、法令に基づく場合、前2項の情報を第三者に対して開示することがあります。
5. 第10条に基づく払戻しの手続等において当社が利用者の個人情報を取得する場合、当社は、当該情報を当該手続及びこれに関する問い合わせの目的にのみ利用し、善良な管理者の注意をもって管理します。

## 第16条（本サービスの変更及び終了）

1. 当社は、本サービスの内容を変更し、又はその全部若しくは一部を終了することがあります。
2. 当社が本サービスの全部を終了する場合、当社は、相当な期間をおいて、当社ウェブサイトへの掲載その他適切な方法により利用者に周知します。

## 第17条（免責及び責任の制限）

1. 当社は、システムの障害、通信回線の障害、カードの故障・破損、停電、天災その他当社の責めに帰することができない事由により本サービスを利用できなかったことによって利用者に生じた損害について、責任を負いません。
2. 当社が利用者に対して損害賠償責任を負う場合、当社の故意又は重大な過失による場合を除き、その賠償の範囲は、利用者に現実に生じた通常かつ直接の損害に限るものとし、特別の事情から生じた損害（逸失利益を含みます。）については責任を負いません。

## 第18条（本規約の変更）

1. 当社は、次のいずれかに該当する場合、本規約を変更することができます。
   1. 変更が利用者の一般の利益に適合するとき
   2. 変更が本規約に係る契約の目的に反せず、変更の必要性、変更後の内容の相当性その他の変更に係る事情に照らして合理的なものであるとき
2. 当社は、本規約を変更する場合、変更後の本規約の内容及びその効力発生時期を、当社ウェブサイトへの掲載その他適切な方法により、その効力発生時期が到来するまでに周知します。

## 第19条（反社会的勢力の排除及び利用資格の取消し）

1. 当社は、利用者が次の各号のいずれかに該当すると認めた場合、直ちに本サービスの利用資格を取り消すことができます。この場合、当社は、事前の通知催告を要せず、本サービスの利用を中止することができます。
   1. 本規約に違反した場合
   2. 反社会的勢力である、又はその疑いがあると当社が合理的な根拠に基づき判断した場合
   3. 本サービスが犯罪に利用されている、又は利用された疑いがあると当社が合理的な根拠に基づき判断した場合
   4. 当社又は加盟店の従業員等に対し、暴力、脅迫、暴言、著しく不当な要求その他その安全若しくは精神衛生を害するおそれのある行為を行った場合
   5. その他本サービスの利用者として不適格と当社が合理的な根拠に基づき判断した場合

## 第20条（準拠法及び合意管轄）

1. 本規約の準拠法は日本法とします。
2. 本規約又は本サービスに関して当社と利用者との間に紛争が生じた場合、東京都千代田区を管轄する簡易裁判所又は地方裁判所を第一審の専属的合意管轄裁判所とします。
